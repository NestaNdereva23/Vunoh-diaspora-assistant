# Vunoh Diaspora Assistant

## Overview
 
Kenyans in the diaspora currently rely on informal channels  WhatsApp messages to relatives, phone calls, word of mouth  to manage tasks back home. These channels are slow, unreliable, and leave no audit trail when things go wrong.
 
This application gives diaspora customers a single interface to:
- Submit a request in plain English
- Get an AI-extracted, structured task with a unique reference code
- Receive confirmation across three channels: WhatsApp, Email, and SMS
- Track task progress through a live dashboard
